# Lazy-Kaoyan-Library
## 项目简介

本项目由一名不知名的26考研生编写。

本项目不是一个静态的笔记仓库，是专为不想手动刷题的人准备的`思维导图`+`错题仓库` (~~不过由于版权原因所有题目无法上传~~

- 前期用 AI 与 OCR 工具辅助转录课程知识与题目成 md 形式
- 后期用 **思维导图 +   AI 对话** 作为入口，串起复习流程；
- 用 **本地优先、无纸化、可持续迭代、双向链接** 的方式代替大量纸质整理。

目前仓库主体包括：

- **数一**：高数、线代、概率论，以及题目整理；
- **英一**：大小作文相关模板与内容；
- **模板文件夹**：基础知识、方法、错题、同类题目汇总等模板；

## 适用对象

如果你本来就喜欢：

- 用 AI 偷懒，自己能少刷题就少刷；
- 无纸化学习，尽量少写重复性的纸质笔记；
- 让知识点和题目形成网络，所有笔记和题目 ALL in one；
- 更注重整理知识库过程中的充实感而非纠结于实际作用与结果的人

那这套仓库思路大概率会很对胃口。


## 项目特色

### 1. AI 友好的本地结构化知识库

- 按“**章节 -> 知识点 -> 方法 -> 题型 -> 题目/错题**”来组织，便于检索、回顾和复盘。
- 可以转录的资料尽量本地 md 化，AI 友好
- 不能转录的资料用图片形式按 `./fig` 路径储存在当前笔记所在文件夹，迁移、同步、清理都更直接。
- 鼓励把所有有价值的题目喂给 ai，卡片笔记形式储存在本地方便回顾，少刷题，多回顾复盘总结

### 2. 思维导图化整理

- 选择思维导图作为入口与复习中心串联卡片笔记，结构与松散动态统一，方便回顾

### 3 . 鼓励动态更新与查询
- 鼓励使用 dataview 动态查询 (~~虽然做的很粗略~~

### 4. 模板驱动

- 鼓励使用 Templater模板，便于持续扩写

## 如何使用

1. 安装 **Obsidian 桌面端**；
2. 下载或克隆本仓库；
3. 在 Obsidian 中选择“打开本地文件夹作为仓库”；
4. 允许加载仓库自带的 `.obsidian` 配置；
5. 按下文的插件建议补齐必要依赖；
6. 从 `数一`、`英一`、`模版文件夹` 开始使用。


## 环境与插件依赖

### 必须安装

- **Templater**：模板文件夹中的模板依赖
- **Dataview**： 动态查询维护
- **Style Settings**：用于配合主题和样式细调；
- **Paste Image Rename**：用于图片粘贴后重命名和管理。

### 推荐安装

- **Obsidian Mind Map NextGen**：用于思维导图类笔记；
- **Excalidraw**：适合后续自己扩展图解、手绘和知识结构图；
- **Easy Typing**：中文输入体验更顺手；
- **Floating TOC**：长笔记目录浏览更方便；
- **Mousewheel Image Zoom**：查看图解时更舒服；
- **OZ Clear Unused Images**：清理无用图片；
- **Obsidian Git**：做版本管理和同步时很省心；
- **Callout Manager**：一键管理所有 Call out
- **Editing-toolbar**：整合常用书写工具，obsiidna word化
- **Floating -toc**：悬浮目录，快速定位
- **Claudian**：直接接入 AI 对话，强烈安利

## 关于图片与附件管理

仓库当前附件路径配置为：

```text
./fig
```


- 图片通常放在当前笔记所在目录下的 `fig` 文件夹；

推荐搭配：

- **Paste Image Rename**：粘贴图片后顺手重命名；
- **OZ Clear Unused Images**：定期清理无引用图片。

## 写在最后面的话

这是最好的时代，这是最坏的时代。

这个时代好在我们有各种开源的工具、资料、经验以及 AI 工具辅助人类学习；

这个时代悲哀在 AI 已经发展到高考考研乃至奥赛都能轻松高分的现在，世界上还有无数人为了一个 N 年后的预期而透支自己的生命在一个个有标准答案的考试上。

我还是坚持认为，如果一个考试事先知道答案再回答就能全对的话，这个考试就是没有意义的。

但世界上还有许多没有答案的东西，这是从有标准答案的考试上学不到的。

本仓库设立的初衷就是借助 AI 工具，减少学生在整理笔记与刷题上的时间，把更多的时间从书本上转移到现实世界与个人生活中。

考研这一年笔者一个很大的收获就是学会了用更成熟的方式表达自己的意见和不满。

这个仓库就是我想说的话。

每个人有每个人的生活方式，可以不认同，但要相互尊重彼此的选择。

祝所有考研学子们能都写下自己的答案。
